# Passenger Counter

A super simple web app that lets you count passengers getting on or off.

## Features

-  Add passengers  
-  Subtract passengers  
-  Live count display  
-  Reset to zero

## Demo
Check out the live demo here: [Passenger Counter Demo](https://yasminserag08.github.io/passenger-counter/)

## Built With

- HTML  
- CSS  
- JavaScript
